Functional requirements:
When the user clicks `Run Program`, the JavaScript should:

1. Perform **Lexical Analysis**

   * Tokenize the input code.
   * Display tokens as chips or in a token table.
   * Recognize:

     * Keywords: let, add, apply, coupon, set, checkout, if, else, for, while, class, new, true, false
     * Identifiers
     * String literals
     * Number literals
     * Operators
     * Symbols
     * Commands

2. Perform **Syntax Checking**

   * Check missing semicolons.
   * Check balanced quotes.
   * Check valid command format.
   * Check variable declaration format.
   * Check add command format:

     * `add "Product Name" quantity @ price;`
   * Check coupon format:

     * `apply coupon "SAVE10";`
   * Check shipping format:

     * `set shipping = 40.00;`
   * Check checkout command:

     * `checkout;`

3. Perform **Semantic Checking**

   * Product must exist in inventory.
   * Quantity must be greater than 0.
   * Price must be valid.
   * Coupon must be valid.
   * Variables should be stored in the variable table.
   * Show semantic errors if something is wrong.

4. Execute the program visually:

   * Add products to cart.
   * Apply coupon.
   * Set shipping.
   * Compute subtotal, discount, and final total.
   * Update the cart, checkout summary, discount status, receipt preview, variable table, and output logs.
